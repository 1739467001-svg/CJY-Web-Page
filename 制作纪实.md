## 项目速览

- **是什么**：陈俊烨（CJY / Vibe Coder）的单页个人站。**新野兽主义（neo-brutalism）**风格——奶油纸底色、墨黑描边、硬阴影、红/蓝/酸橙撞色、超大海报字，并以 🦞 **小龙虾**为贯穿全站的 IP。
- **技术栈**：纯静态，**零运行时依赖**。手写 HTML + CSS + 原生 JavaScript；动效用 Web Animations API + `requestAnimationFrame`；部署用 nginx + Docker（Zeabur 自动识别）。
- **体量**：**21 次提交** · 跨度约 **30 小时** · 3 个核心文件约 **2130 行**（`index.html` 395 / `styles.css` 777 / `main.js` 958）· **0 依赖**。
- **四条底线（每一步都遵守）**：
  1. 无障碍——`prefers-reduced-motion` 下一切动效归于平静；
  2. 响应式——手机 390 / 平板 768 / 桌面 1080+ 三档都成立，移动端自动缩量；
  3. 动效自清理——粒子/彩蛋用完即移除 DOM，切后台暂停；
  4. 不破坏可用性——原生光标保留、按钮不被遮挡、无横向溢出。

## 我们是怎么协作的（可复用的方法）

1. **对话式迭代，小步提交**：一次只推进一个清晰目标，立刻 commit，message 写清"做了什么 / 为什么 / 怎么验证"。21 次提交就是 21 个可回溯的节点。
2. **每个视觉改动都用无头浏览器截图自检**：不是"我觉得行"，而是真把它渲染出来看——还会量元素盒子坐标、查是否溢出、查是否挡住按钮、数彩蛋是否被吃掉、确认动画结束后 DOM 清零。
3. **两条硬约束贯穿始终**：无障碍（reduced-motion 必须能彻底安静）＋ 响应式（三档都要成立、移动端缩量）。
4. **动效自清理**：所有粒子/彩蛋用完即移除，切后台暂停 rAF，绝不泄漏。
5. **零依赖**：能用原生就不引库，保持站点轻、可控、好部署。
